# Heap Insertion Time Analysis

## Project Overview
This project is based on Data Structures and focuses on analyzing the time complexity of insertion operation in a Heap (Binary Heap).

## Objective
The objective of this project is to understand how heap insertion works and how its performance behaves in terms of time complexity.

## Key Concepts
- Binary Heap (Min Heap / Max Heap)  
- Insertion Operation  
- Time Complexity Analysis  
- Tree Structure and Heap Property  

## Working Explanation
In this project, elements are inserted into a heap data structure while maintaining the heap property. After each insertion, the element is compared with its parent and adjusted accordingly (heapify process).

## Time Complexity
The time complexity of heap insertion is:

O(log n)

because in the worst case, the element moves from the bottom to the root of the heap.
